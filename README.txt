This plugging sends a message to the users' facebook account once an update on mooldle quizes are made, if he /she has subscribed to the service.
The message sanding may get affected according to the server you used. In some cases Facebook will not allow to send messages through the server that you are using. 


Installation (Before installing the plugging please read the following instructions)
----------------------------------------------------------------------------------------
1) This plugin supports for Moodle 2.4.4. and upper

2) Place the pluging inside the moodle/block folder

3) Login to the system under the Administration privillages

4) Upgrade the system

Thank you for using the plugin !!!
